┌────────────────────────────────────────────────────────────────┐
│  POUR LES UTILISATEURS (Utiliser le .exe)                     │
└────────────────────────────────────────────────────────────────┘

1. Double-cliquez sur "LLM_Checker.exe"
2. L'application détecte automatiquement votre hardware
3. Consultez les 5 meilleurs modèles Ollama pour votre PC
4. Changez le "Use Case" selon vos besoins
5. Copiez les commandes "ollama pull" pour installer les modèles

✅ Aucune installation requise!
✅ Fonctionne sur Windows 7/8/10/11


┌────────────────────────────────────────────────────────────────┐
│  POUR LES DÉVELOPPEURS (Créer le .exe)                        │
└────────────────────────────────────────────────────────────────┘

═══ MÉTHODE 1: AUTOMATIQUE (Recommandé) ═══

1. Double-cliquez sur "build_exe.bat"
2. Suivez les instructions à l'écran
3. Le .exe sera créé dans le dossier "dist/"

═══ MÉTHODE 2: MANUELLE ═══

1. Ouvrir un terminal (CMD ou PowerShell)

2. Naviguer vers le dossier python_gui:
   cd chemin\vers\llm-checker\python_gui

3. Installer les dépendances:
   pip install -r requirements.txt

4. Tester l'application:
   python llm_checker_gui.py

5. Créer le .exe:
   pyinstaller llm_checker.spec

6. Le .exe sera dans: dist\LLM_Checker.exe

┌────────────────────────────────────────────────────────────────┐
│  RÉSOLUTION DE PROBLÈMES                                       │
└────────────────────────────────────────────────────────────────┘

❌ "python n'est pas reconnu"
   → Réinstallez Python et cochez "Add to PATH"

❌ "ModuleNotFoundError: No module named 'psutil'"
   → pip install psutil

❌ Le .exe ne se lance pas
   → Vérifiez dans dist/LLM_Checker.exe
   → Testez sans --windowed pour voir les erreurs

❌ Antivirus bloque le .exe
   → Normal pour PyInstaller, ajoutez une exception
